Enumerable Methods!

11-21-2015

We've learned this week how useful enumerable methods can be, in particular, with iterating. Meaning they go through a collection of elements and "do" something to each of those elements.

I'll be further explaining a couple examples here. Enumerable method "Count", which is to safe to assume, will count the number of things in the given collection. Here's a bit more complex enumerable method: "each_slice", which will iterate through and "slice" every requested number of elements, where you can pass in a block to "do" something at every slice. Below is an example of how you can "slice" a group into groups of 3 and return them as an array. 

def groups_of_3(argument)
        array = []
        argument.each_slice(3) { |x| array.push(x) }
        array
end
