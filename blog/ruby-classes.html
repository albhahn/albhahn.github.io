Have some class!

11-28-2015

This week, we got to learn how to create a class and everything that comes with it, like an instance variable. You can think of a class like a blueprint from which individual objects are created. For instance, a String is a class for "Hi" or "word" and a Fixnum is a class for a number.

First thing you want to do when creating a class, is define it. You can do so like you would "def" a method, the main difference is you must capitalize the class Name. Once you name your class, you need to define a method "initialize", which is the constructor. This allows you to create a new instance of your class. We can now define other instance methods to do something for our class. If you look at my example below, I've created the class "Dog", defined a method "initialize" which will take in a name for my dog, and created a method for my dog to "bark". 

class Dog
  def initialize(name)
    @name = name
  end

  def bark
    puts "Ruff-Ruff!"
  end
end

Now the next example below will show you how to create a new instance of dog, and have that dog "bark". Lets say my dogs name was "Fido": 

fido = Dog.new("Fido")
fido.bark

There are a ton of things you can do with classes and be very flexible with instance methods/variables. Although this was a small example, I hope this was helpful to you in some way of undestanding how classes work. Practice makes perfect and coding is no exception, so my best advice with classes is to simply practice writing them and instantiating between them!
